## How to put in Microsoft Word

1. Place the cursor where the figure should go (replace the placeholder text in the generated report if needed).
2. **Insert → Pictures → This Device** and choose the `.svg` file (Word 2016 and later supports SVG).
3. Click the image → **Picture Format** → set **Wrap Text** to **In line with text** (keeps alignment with paragraphs).
4. Resize width to about **15–16 cm** (or 6–6.3 inches) so it fits the page margins cleanly.
5. Add the caption below: **References → Insert Caption** (e.g. “Figure 1: …”).

If SVG does not insert correctly on your Word version, use the `*_vector.pdf`: **Insert → Object** or insert as image after converting PDF page to PNG via snipping tool.
